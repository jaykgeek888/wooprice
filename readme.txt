Simple plugin for displaying woocommerce product price by productID and shortcode.

== Description ==
You can display your product price on your woocommerce shop by a shortcode.

== Installation ==
1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress

OR install with wp admin dashboard

1. go to admin>plugins>add
2. brows & select woo-display-price.zip files
3. install & active plugin
